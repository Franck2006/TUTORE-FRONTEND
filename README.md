voici les commandes git à savoir 
- [ git clone https:....git : pour cloner le site ]
- [ git add index.html: pour ajouter la fiche HTML qui a commu une modification ]
- [ git add . : pour ajouter toutes les fiches en generale qui ont commu une modification ]
- [ git commit -m "msg personal" : pour associer un msg à une modification ]
- [ git push : pour envoyer le code au dossier en ligne sur gitHub ]
